До якого типу алгоритмів належить даний приклад програмного коду?

A = [2, 5, 14, 62, -5]


for i in range(len(A)):

    if A[i]<0:

        A[i] *= 2


print(A)


# Результатом буде: [2, 5, 14, 62, 0]

Правильно! ✅

Не правильно... ❌

До якого типу алгоритмів належить даний приклад програмного коду?

A = [2, 0, 14, 62, -5]


k = 0

for i in A:

    if i > 0:

        k += 1


print(k)


# Результатом буде: 3

Правильно! ✅

Не правильно... ❌

До якого типу алгоритмів належить даний приклад програмного коду?

A = [2, 0, 14, 62, -5]


mx = A[0]

for i in range(1, len(A)):

    if A[i] > mx:

        mx = A[i]


print(mx)


# Результатом буде: 62

Правильно! ✅

Не правильно... ❌
